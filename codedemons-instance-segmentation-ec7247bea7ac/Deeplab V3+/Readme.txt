The above file contains the implentation of Deeplabv3+ for semantic segmentations on Indian Driving Dataset.
The dataset can be downloaded from the following link https://idd.insaan.iiit.ac.in/accounts/login/?next=/dataset/download/
We have trained our model using IDD lite dataset a subsampled dataset of IDD.
the dataset has 5000 images and the labels are in the form of masked images

The backbone CNN model used here is Resnet50
